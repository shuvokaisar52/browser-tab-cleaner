# 🧹 CleanTab - Smart Tab Cleaner

Browser Tab Cleaner is a lightweight browser extension that automatically closes inactive tabs after a configurable period of time. It helps reduce tab clutter, improve organization, and keep your browsing environment clean and manageable.

---

## 🚀 Features

- ⏱️ Automatically closes inactive tabs after a set duration  
- 🎯 Tracks activity per tab (focus, interaction, navigation)  
- ⚙️ Fully configurable timeout settings  
- 🧩 Lightweight background processing  
- 🔒 Runs locally in the browser (no external servers or data collection)  

---

## 🧠 How It Works

1. When a tab becomes inactive, a timer starts  
2. If no interaction occurs within the configured time limit, the tab is closed  
3. Any user activity resets the timer  
4. Active tabs remain unaffected
5. Closed tabs are preserved in browser history (can be reopened later)
